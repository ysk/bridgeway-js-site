実装サンプル

まずはjQueryの書き味そのままから。実務でよくある処理を、いつもの手で——ただし合計・絞り込み・出し分けは state／computed が自動で追従します（手で書き直さない）。埋め込みは実際に動くデモです（ローカルは node build.demo.mjs 後に表示）。

$ → $$ にするだけ

既存のjQueryコードは、ほとんど $ を $$ に置き換えるだけで動きます。手が覚えた書き方はそのまま。

jQuery
$(".tab").on("click", function () {
  $(".tab").removeClass("active");
  $(this).addClass("active");
});
$("#save").prop("disabled", true).text("送信中…");
$(".alert").fadeOut();
$(e.target).closest("tr").remove();
bridgey
$$(".tab").on("click", function () {
  $$(".tab").removeClass("active");
  $$(this).addClass("active");
});
$$("#save").prop("disabled", true).text("送信中…");
$$(".alert").fadeOut();
$$(e.target).closest("tr").remove();
チェーン・イベント委譲・this・fadeOut・closest… jQueryの手癖はそのまま使えます。

使えるメソッド

jQuery定番をひととおり実装しています。

リアクティブ束縛: bindText bindClass bindAttr bindValue ／ フォーム: serialize serializeArray ／ 通信: $$.ajax $$.get $$.post(.done/.fail/.always) ／ 静的: $$.extend $$.each $$.map $$.trim ／ 自作拡張: $$.fn。詳細は Docs。

既存のjQueryプラグイン（slick等）はそのまま動きません。$$.fn は自作プラグイン用です。既存プラグインは領域を分けて本物のjQueryを併用してください。

フォームバリデーション（実務）

入力ごとの検証・エラー表示・パスワード強度・文字数カウンタ・「全項目OKで送信有効」。検証は $$ の書き味、送信可否は state/computed で宣言的に。送信は $$.post（fetch）で実API風に。

Contact form

pure $$ + state 開く
// 入力のたびに検証（jQueryそのまま）
$$("#email").on("input blur", function () {
  const msg = /^[^@\s]+@[^@\s]+\.[^@\s]+$/.test(this.value)
    ? "" : "メール形式が正しくありません";
  $$("#err-email").text(msg);
  $$(this).toggleClass("invalid", !!msg);
});

// 「全項目OK」は computed で宣言的に → 送信ボタンに束縛
const valid = computed(() => ok(form.value));
valid.subscribe((v) => $$("#submit").prop("disabled", !v));

タブ・合計・アニメ（基本）

タブ切り替え（addClass/removeClass＋委譲）、リアルタイム合計（input → state → computed）、アコーディオン（slideToggle）とトースト（fadeIn/fadeOut）。命令的と宣言的を1画面で。

Tabs & total

pure $$ + state 開く
// タブ（jQuery定番: 委譲 + addClass/removeClass）
$$("#tabs").on("click", ".tab", function () {
  $$("#tabs .tab").removeClass("active");
  $$(this).addClass("active");
});

// リアルタイム合計（input → state → computed → 表示）
const total = computed(() => price.value * qty.value);
$$("#total").bindText(total, (v) => "¥" + v);

ライブ検索フィルタ（実務）

入力のたびに一覧を絞り込み、ヒット箇所を <mark> で強調。分類チップとのAND条件も computed で宣言的に。検索語・分類はそれぞれ state、絞り込み結果は computed が導きます。

Live search

pure $$ + state 開く
// 検索語と分類はそれぞれ state に
const query = state(""), cat = state("all");

// 絞り込みは computed（依存が変われば自動で再計算）
const filtered = computed(() => {
  const q = query.value.trim().toLowerCase();
  return items.filter((it) =>
    (cat.value === "all" || it.cat === cat.value) &&
    (q === "" || it.name.toLowerCase().includes(q)));
});
filtered.subscribe(render); // 変わるたび一覧を描き直す

// 入力は命令的に受ける（jQueryそのまま）
$$("#q").on("input", function () { query.value = this.value; });
$$("#chips").on("click", ".chip", function () {
  $$("#chips .chip").removeClass("active");
  $$(this).addClass("active");
  cat.value = this.dataset.cat;
});

ショッピングカート（実務）

数量の増減・行小計・クーポン・送料無料判定・合計。金額はすべて computed で、状態が変わるたび自動再計算。操作はイベント委譲＋closest＋fadeOut の jQuery 定番です。

Shopping cart

pure $$ + computed 開く
// 金額は積み上げず computed で導く（cart / coupon に依存）
const subtotal = computed(() => cart.value.reduce((s, it) => s + it.price * it.qty, 0));
const discount = computed(() => Math.floor(subtotal.value * (coupon.value?.rate ?? 0)));
const total    = computed(() => subtotal.value - discount.value + shipping.value);

// 増減・削除はイベント委譲（再描画されても効く。this=クリック要素）
$$("#cart")
  .on("click", ".plus", function () { patch(idOf(this), (it) => ({ ...it, qty: it.qty + 1 })); })
  .on("click", ".del",  function () { $$(this).closest(".item").fadeOut(150); /* …state から除去 */ });

// サマリー行は computed に束縛（bindText で自動追従）
$$("#total").bindText(total, (v) => "¥" + v.toLocaleString());

モーダル・画像ギャラリー（実務）

タイルをクリックで拡大するライトボックス。イベント委譲で開き、fadeIn/fadeOut で開閉、背景クリック・Esc・矢印キーにも対応。「今どれを開いているか」だけ state で持ちます。

Lightbox gallery

pure $$ + state 開く
// 開いている index だけ state。前へ/次へで付け替える
const current = state(-1);
current.subscribe((i) => { if (i >= 0) fillLightbox(photos[i]); });

// タイルを開く（委譲：後から足した要素にも効く）
$$("#gallery").on("click", ".tile", function () {
  current.value = Number($$(this).data("i"));
  $$("#lb").fadeIn(150);
});
// 背景クリックで閉じる（オーバーレイ自身を押したときだけ）
$$("#lb").on("click", function (e) { if (e.target === this) $$("#lb").fadeOut(150); });
$$(document).on("keydown", (e) => { if (e.key === "Escape") close(); });

星評価・レビュー（実務）

ホバーでプレビュー、クリックで確定。平均・分布バー・投稿一覧はすべて state / computed。星の塗り分けは $$ の .each と .toggleClass で。

Star rating & reviews

pure $$ + computed 開く
// 星の塗りは jQuery の .each そのまま（n個ぶんを on に）
function paint(n) {
  $$("#input-stars .star").each(function () {
    $$(this).toggleClass("on", Number(this.dataset.v) <= n);
  });
}
// ホバーは仮表示・クリックで確定値(state)を更新
$$("#input-stars")
  .on("mouseover", ".star", function () { paint(Number(this.dataset.v)); })
  .on("mouseleave", () => paint(myScore.value))
  .on("click", ".star", function () { myScore.value = Number(this.dataset.v); });

// 平均は computed → bindText で自動追従
const average = computed(() => avg(reviews.value));
$$("#avg").bindText(average, (v) => v.toFixed(1));

命令的 → 宣言的（段階移行）

bridgeyの芯。まずは命令的（jQuery脳のまま手で更新）。慣れたら宣言的（状態を変えるだけで自動反映）へ、同じ画面で少しずつ。

① 命令的（いつもの手）
let n = 0;
$$("#inc").on("click", () =>
  $$("#count").text(++n)   // 手で更新
);
② 宣言的（自動追従）
const n = state(0);
$$("#inc").on("click", () => n.value++);
$$("#count").bindText(n); // 自動

レガシー×Svelteを state で橋渡し（卒業ルート）

全書き換えは要りません。ページを領域で色分けして、入力・イベントは jQuery の領域のまま、複雑な描画だけSvelteに受け持たせた領域に切り出す。両者は共有 state で会話します——お互いのDOMには触らないのでズレません。1ページの中で、少しずつモダンへ卒業していけます。

Legacy × Svelte

$$ + state + svelte 開く
// 共有 state = 橋（どちらのDOMにも相乗りしない）
const query = state("");
const results = computed(query, (q) => q ? ALL.filter((x) => x.includes(q)) : ALL);

// jQueryの領域: DOMを直接いじらず state に書くだけ
$$("#q").on("input", (e) => (query.value = e.target.value));

// Svelteの領域: 共有 state を渡して描画を丸投げ
const app = mount(App, { target: "#panel", props: { query, items: results } });

// Svelte → jQuery: Svelteが出したイベントを jQuery で受ける
app.on("pick", (e) => $$("#picked").text("選択: " + e.detail));
逆に、Svelteが描画したノードを $$ で直接書き換えるのは避けてください（再描画でズレます）。会話は必ず state かイベント経由で。

本物の部品を載せる（Svelte）

複雑な画面は本物のSvelteへ。mount で載せるだけ。

Counter component

svelte 開く
import { mount, useEngine } from "bridgey";
import { svelteEngine } from "bridgey/engines/svelte.js";
useEngine(svelteEngine);
const app = mount(App, { target: "#app" });
